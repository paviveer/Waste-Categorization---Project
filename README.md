# Waste-Categorization---Project
## 📂 Dataset

The dataset is large and cannot be uploaded to GitHub.

You can download it from Google Drive:

👉https://drive.google.com/drive/folders/19m2771MoxyXFTyB1lTfgUoNwCvF3o5ZI?usp=sharing
After downloading, place the dataset folder inside the project directory before running the notebook.
⚠️ Note:
The notebook was developed in Google Colab.
If running locally, update the dataset path inside the notebook according to your system.

**How to Run**
Install required libraries:

pip install -r requirements.txt

Open the notebook in Google Colab or Jupyter.

Make sure:
Dataset folder is placed correctly.
Dataset path inside notebook is correct.
